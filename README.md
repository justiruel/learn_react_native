# State
Semacam session yang pendefinisiannya di dalam class

SET
===

~STATIC~
this.state ={
      nama:'irul'
}

~DYNAMIC~
this.setState({nama:text})

GET
===
{this.state.nama}


# Props 
Semacam session yang pendefinisiannya di luar class

SET
===
<Object_gue nama="joni"/>

GET
===
{this.props.nama}
